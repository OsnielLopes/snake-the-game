Jogo Snake desenvolvido para a disciplina de Linguagem de Programação I.

Necessário Python 3.4 e plugin pygame.

Verifique o arquivo jogodacobrinha.pptx para acessar o manual do usuário e aprender a jogar.

Código pode ser editado pelo Python 3.4 Compiler abrindo o arquivo .py.

Para movimentar a cobra, o usuário deve apertar as teclas WASD, ou as setas.
